Introduction

Now that we are comfortable zooming around Excel without a mouse, we need to get comfortable entering information (it's kind of the point). In this lesson, we'll cover the basics of entering information quickly.

Before we get started, you'll want to download the files below for this lesson:

  • Exercises for this lesson
  • Answers for this lesson

Lesson 1: Entering information vertically

Usually, after you are done putting some information in a cell, you will move on to another cell to enter more information. Not surprisingly, the Enter / Return key is what you press when you're done entering information and looking to information in a line below your current line.

Try entering some information in a cell, press return, and see where your cursor moves in Excel. Once this makes makes complete sense, go ahead and complete the exercise.

Lesson 2: Entering information horizonttaly

Now that you can enter as much information going down as you want, you'll want to start moving right. To do this, after you're done with a cell, use the TAB key to move right.

Spend some time putting this in your memory bank and complete the exercise once you're ready.

Lesson 3: Entering information in all directions

All right, so I lied a little. As cool as Return and Tab are for entering information, they only move downwards or rightwards respectively. Sometimes, you'll want to move backwards. There are two ways to do this:

  1. You can hold SHIFT while pressing Return or Tab, to move up or left respectively
  2. Or you can just use the Up or Left arrow keys

An aside on using Up and Left (and just down and right) is that in certain scenarios (e.g. when editing formulas) they do not work. This is the reason to learn Return and Tab, as they always work for completing information entry.

Try these shortcuts out for a little and complete the exercise!

Lesson 4: Deleting data

Now that you've created, you must learn to destroy. To do this, we want to use the Delete key. Depending on the version of Excel and what configuration of keyboard you have, there is a difference in functionality between Delete and Backspace, so make sure you try both of them to understand the differences.

For this exercise, we'll revisit some of the multi-select and skipping cell lessons in the previous section to delete information in multiple cells at once. Make sure to keep those shortcuts in mind, so you aren't deleting every cell individually. And to get some repetition, undo your deletes (CTRL + Z on Windows / COMMAND + Z on Mac) a few times to make sure you are have this committed to memory.

The Challenge

Now that we have learned to add and remove data from cells, we will practice a bit more with some math. In this section's challenge, we will build the multiplication table.

Currently, there are letters going across and downwards in the grey cells. We obviously can't multiply letters, so replace the alphabet with the numbers 1 through 9 both horizontally and vertically and see what happens! Remember to only use your keyboard for this exercise.
